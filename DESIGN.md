# ACT Console v2 设计规范

> 基于 Home Console v1 重构，以 ACT 三层框架为核心组织 Tab 结构
> 最后更新: 2026-06-13

---

## 1. 与 v1 (Home Console) 的核心区别

### Tab 结构变化

| v1 Tab | v2 Tab | 变化说明 |
|--------|--------|---------|
| 行动指北 | **今日聚焦** (默认) | ACT 健康指标 + 滴答日程 + 本周要事 + 周计划 + 12周流程 + 日志 |
| 任务推进 | **行动 A** | 左：任务推进看板（原任务推进），右：逾期/临期追踪（原行动指北） |
| _(无)_ | **知识 C** _(新增)_ | 知识层总览（四类卡片数）+ 最近写的卡片 + 待处理材料 + 资料入口 |
| _(无)_ | **方向 T** _(新增)_ | 12 周目标进度条 + 愿景 + 周记历史 + 日志热力图 |
| 内容创作 | 内容创作 _(可选)_ | 设置中开启/关闭 |
| 数据分析 | 数据分析 _(可选)_ | 设置中开启/关闭 |
| 稍后阅读 | 稍后阅读 _(可选)_ | 设置中开启/关闭 |
| AI 会话 | AI 会话 _(可选)_ | 设置中开启/关闭 |

### 设计原则

1. **ACT 框架优先**：四个核心 Tab 直接对应课程教学的三层结构 + 每日焦点
2. **课程学员友好**：默认只显示 ACT 核心 Tab，进阶用户在设置中开启可选 Tab
3. **向后兼容**：保留 v1 所有功能代码，只重新组织 Tab 归属
4. **独立部署**：插件 ID 为 `act-console`，不影响 v1 的 `kivenbig-home-console`

## 3. 可选 Tab 机制

设置项 `enabledOptionalTabs: OptionalTab[]`，四个可选值：
- `content` → 内容创作
- `analytics` → 数据分析  
- `read` → 稍后阅读
- `ai` → AI 会话

课程模板默认全部关闭，个人使用默认全部开启。

---

## 4. 新增组件

### ACT 健康指标卡 (`.khc-metric`)

| 指标 | 数据来源 | 颜色 |
|------|---------|------|
| 今日日志 | 30-Time/34-Daily 是否存在今日文件 | 绿/紫 |
| 进行中任务 | 10-Action 下 a1-要事推进 笔记数 | 绿/深色(有逾期) |
| 本周待办 | 周记"本周计划"区块未完成数 | 绿/紫 |
| 周期进度 | 当前周/12 | 紫 |
| 知识卡片 | 核心卡 + 索引卡总数 | 绿 |

### 12 周进度条 (`.khc-cycle-progress`)

- 背景条 + 填充条 + 文字标签
- 深色主题下自适应颜色

---

## 5. 色板与字体

继承 v1 的设计规范，两套主题不变：
- **Payview SaaS**：深绿主色 · 柔和阴影
- **Accounting Desktop**：暖白画布 · 硬偏移阴影

---

## 6. 随想速记整合

随想速记保持独立插件。快捷操作栏"随想"按钮调起随想速记插件命令。
随想与任务的关联在「行动 A → 任务推进」看板中实现。
